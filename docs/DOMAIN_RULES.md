# DOMAIN_RULES

## 1. Produkt nie jest doradcą decyzyjnym
Aplikacja:
- nie wydaje decyzji administracyjnych,
- nie rozstrzyga odpowiedzialności prawnej,
- nie zastępuje kierownika, prawnika ani służb.

## 2. Każda odpowiedź proceduralna powinna mieć:
- krótki opis sytuacji,
- co zrobić teraz,
- kogo powiadomić,
- czego nie pominąć,
- podstawę prawną / źródło,
- informację o eskalacji,
- wymagane dokumenty.

## 3. Dane demo
W demo używamy:
- danych przykładowych,
- neutralnych nazw,
- zero realnych danych klientów.

## 4. Język
- prosty,
- operacyjny,
- bez prawniczego nadmiaru,
- czytelny w stresie.

## 5. Priorytety odpowiedzi
1. bezpieczeństwo dziecka i użytkownika,
2. zgodność proceduralna,
3. kompletność dokumentacji,
4. minimalizacja ryzyka pominięcia kroku.
